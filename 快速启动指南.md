# ⚡ 快速启动指南

## 🎯 5秒钟开始使用

### 在电脑上：
1. 双击打开 `index.html`
2. 开始使用！

### 在手机/iPad上：
1. 浏览器打开 `index-mobile.html`
2. 开始使用！

### 不确定用哪个？
1. 打开 `选择版本.html`
2. 系统自动推荐！

## 🚀 基本操作流程

### 1️⃣ 上传EP图片
```
点击 "📷 上传图片" → 选择图片 → 完成
```

### 2️⃣ 添加经纱
```
点击 "📷 拍照" 或 "🎤 录音" 或 输入文本
需要多个？点击 "➕" 按钮
```

### 3️⃣ 添加纬纱
```
点击 "📷 拍照" 或 "🎤 录音" 或 输入文本
需要多个？点击 "➕" 按钮
```

### 4️⃣ 填写纬密
```
输入实际测量值（根/10cm）
```

### 5️⃣ 记录问题（可选）
```
点击 "🎤 录音" 或 "🎥 录像"
```

### 6️⃣ 记录成品（可选）
```
点击 "📷 拍照" 或 "🎥 录像"
```

### 7️⃣ 增加更多页（重要！）
```
点击底部 "➕ 增加页"
→ 自动复制上一页内容
→ 只需修改不同部分
→ 避免重复输入
```

### 8️⃣ 保存记录
```
点击 "💾 保存所有页"
```

## 💡 快速技巧

### ✅ 快速清空
所有输入框和媒体都有 "×" 按钮，点击即可清空

### ✅ 批量录入
1. 第一页填写完整信息
2. 点击 "增加页"
3. 新页面自动复制
4. 只修改不同部分

### ✅ 删除内容
- **删除页面**：点击页面右上角 "×"
- **删除纱线**：点击纱线的 "删除" 按钮
- **删除媒体**：点击媒体右上角 "×"

## 📱 添加到主屏幕（推荐）

### iOS设备：
1. Safari打开文件
2. 点击底部分享按钮 📤
3. 选择 "添加到主屏幕"
4. 点击 "添加"
5. 完成！像App一样使用

### Android设备：
1. Chrome打开文件
2. 点击右上角菜单 ⋮
3. 选择 "添加到主屏幕"
4. 点击 "添加"
5. 完成！像App一样使用

## 🔧 权限设置

### 首次使用需要允许：
- ✅ 相机权限（用于拍照、录像）
- ✅ 麦克风权限（用于录音）

### iOS设置：
```
设置 → Safari → 相机/麦克风 → 允许
```

### Android设置：
```
设置 → 应用 → Chrome → 权限 → 相机/麦克风 → 允许
```

## 💾 数据说明

### 数据存储位置
- 浏览器本地（IndexedDB）
- 不上传到服务器
- 完全离线使用

### 数据安全
- ✅ 永久保存（除非清除浏览器数据）
- ✅ 隐私保护（仅本地存储）
- ✅ 可导出备份（JSON格式）

## ❓ 常见问题

### Q: 手机和电脑的数据能同步吗？
A: 数据是独立的。如需同步，请使用导出/导入功能。

### Q: 可以离线使用吗？
A: 完全可以！不需要网络连接。

### Q: 数据会丢失吗？
A: 不会！除非清除浏览器数据。建议定期导出备份。

### Q: 为什么需要相机和麦克风权限？
A: 用于拍照、录音、录像功能。
