# 📱 提花织物生产记录系统

一个专为提花织物生产设计的多平台记录系统，支持桌面、手机、iPad等多种设备。

---

## ✨ 特色功能

### 🎯 核心功能
- ✅ EP文件名图片上传（支持微信导出图片）
- ✅ 多个经纱模型记录（拍照、录音、文本）
- ✅ 多个纬纱模型记录（拍照、录音、文本）
- ✅ 实际纬密值记录
- ✅ 生产问题记录（录音、录像）
- ✅ 成品记录（拍照、录像）
- ✅ 多页功能（自动复制上一页内容）
- ✅ 历史记录查看
- ✅ 数据导出

### 🎨 设计特点
- ✅ **苹果风格毛玻璃效果**（移动版）
- ✅ **响应式设计**（自动适配设备）
- ✅ **触摸优化**（按压反馈、平滑动画）
- ✅ **紧凑布局**（节省屏幕空间）
- ✅ **离线使用**（无需网络）

## 🚀 快速开始

### 方法1：直接打开（最简单）

**桌面电脑：**
1. 双击打开 `index.html`
2. 开始使用！

**手机/iPad：**
1. 将文件夹传输到设备
2. 用浏览器打开 `index-mobile.html`
3. 开始使用！

### 方法2：添加到主屏幕（推荐）

**iOS设备：**
1. Safari打开 `index-mobile.html`
2. 点击分享 → "添加到主屏幕"
3. 像App一样使用！

**Android设备：**
1. Chrome打开 `index-mobile.html`
2. 菜单 → "添加到主屏幕"

### 方法3：本地服务器（可选）

```bash
# 启动服务器
python -m http.server 8000

# 浏览器访问
http://localhost:8000
```

## 📖 使用说明

### 基本流程

1. **上传EP图片** → 点击"📷 上传图片"
2. **添加经纱** → 拍照/录音/输入文本
3. **添加纬纱** → 拍照/录音/输入文本
4. **填写纬密** → 输入实际测量值
5. **记录问题** → 录音/录像（可选）
6. **记录成品** → 拍照/录像（可选）
7. **增加页面** → 点击"➕ 增加页"（自动复制）
8. **保存记录** → 点击"💾 保存所有页"

### 快速技巧

✅ **快速清空** - 所有输入框和媒体都有"×"按钮  
✅ **批量录入** - 增加页时自动复制上一页内容  
✅ **多页记录** - 一次可以记录多个产品  
✅ **离线使用** - 不需要网络连接

## 🔧 技术栈

- **前端**：纯HTML + CSS + JavaScript
- **数据库**：IndexedDB（浏览器本地）
- **媒体**：MediaRecorder API

- **兼容性**：现代浏览器（Chrome、Safari、Edge）

## 📞 常见问题

### Q: 数据会丢失吗？
A: 不会！数据保存在浏览器本地，除非清除浏览器数据。建议定期导出备份。

### Q: 可以离线使用吗？
A: 完全可以！不需要网络连接。

### Q: 手机和电脑的数据能同步吗？
A: 数据是独立的。如需同步，请使用导出/导入功能。

### Q: 为什么需要相机和麦克风权限？
A: 用于拍照、录音、录像功能。首次使用时浏览器会询问。
