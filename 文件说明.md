## 🚀 启动文件详解

### 1. index.html（桌面版）
```
用途：桌面电脑使用
特点：宽松布局、鼠标优化
适合：办公室、笔记本电脑
打开：双击文件即可
```

### 2. index-mobile.html（移动版）
```
用途：手机和iPad使用
特点：自动检测设备、加载对应样式
适合：iPhone、Android、iPad
打开：浏览器打开即可
智能：
  - 手机 → 自动加载 styles-mobile.css
  - iPad → 自动加载 styles-ipad.css
```

```
用途：不确定用哪个版本时使用
特点：自动检测设备并推荐
功能：

  - 推荐最适合的版本
  - 可手动选择版本
```

## 🔧 功能文件详解

### 1. database.js（数据库管理）
```javascript
功能：
  - IndexedDB初始化
  - 保存记录
  - 读取记录
  - 删除记录
  - 搜索记录
  - 导出数据

数据结构：
  {
    id: 自动生成,
    timestamp: 时间戳,
    pages: [
      {
        epImage: EP图片,
        warpYarns: 经纱数组,
        weftYarns: 纬纱数组,
        actualDensity: 实际纬密,
        problems: 问题数组,
        products: 成品数组
      }
    ]
  }
```

### 2. media.js（媒体处理）
```javascript
功能：
  - 拍照（调用相机）
  - 录音（MediaRecorder）
  - 录像（MediaRecorder）
  - 媒体预览
  - Base64编码

支持格式：
  - 图片：JPEG、PNG
  - 音频：WebM、MP4
  - 视频：WebM、MP4
```

### 3. app.js（应用逻辑）
```javascript
功能：
  - 页面管理（增加、删除、复制）
  - 纱线管理（经纱、纬纱）
  - 媒体管理（上传、删除）
  - 数据保存
  - 历史记录
  - 搜索筛选

核心类：
  class ProductionRecordApp {
    - pages: 页面数组
    - pageCount: 页面计数
    - addPage(): 增加页面
    - deletePage(): 删除页面
    - saveAllPages(): 保存所有页
    - ...
  }
```
