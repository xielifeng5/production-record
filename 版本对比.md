## 🔄 自动切换逻辑

移动版（`index-mobile.html`）会自动检测设备：

```javascript
function detectDevice() {
    const width = window.innerWidth;
    const height = window.innerHeight;
    
    // iPad 12.9寸检测 (1024x1366 或更大)
    if ((width >= 1024 && height >= 1366) || 
        (width >= 1366 && height >= 1024)) {
        return 'ipad';
    }
    
    // 其他设备使用手机版
    return 'mobile';
}
```

## 🎨 视觉效果预览

### 毛玻璃效果特点：
1. **半透明背景** - 可以看到后面的渐变色
2. **模糊效果** - 背景内容被模糊处理
3. **饱和度增强** - 颜色更鲜艳（saturate 180%）
4. **层次感** - 多层卡片叠加，立体感强
